# MINIMA - Tienda de Moda Minimalista

Aplicación de tienda de moda moderna y minimalista construida con React, TypeScript y Tailwind CSS.

## Características
- Diseño minimalista con paleta de beige y gris oscuro.
- Tipografía Montserrat.
- Navegación fluida entre productos y detalles.
- Autenticación con Supabase.
- Integración de pagos con PayPal.

## Tecnologías
- **Frontend**: React (Vite) + TypeScript
- **Estilos**: Tailwind CSS
- **Autenticación**: Supabase
- **Pagos**: PayPal SDK
- **Iconos**: Lucide React
- **Animaciones**: Framer Motion

## Instalación

1. Clona el repositorio.
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Configura las variables de entorno:
   Crea un archivo `.env` basado en `.env.example` y añade tus claves de Supabase.
4. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```

## Pruebas
Para ejecutar las pruebas unitarias:
```bash
npm test
```
